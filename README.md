# AmiConnect Backend

## Setup Instructions

1. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```
2. **Configure environment**
   - Copy `.env.example` to `.env` and fill in your database credentials and secret key.
3. **Run the app**
   ```bash
   python app.py
   ```
